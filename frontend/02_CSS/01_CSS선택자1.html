CSS 선택자 1

선택자란?
특정 HTML 태그를 선택할 때 사용한다.
태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

1. 전체 선택자(*)

HTML 문서 내부에 모든 태그를 선택할 때 사용한다.

2. 태그 선택자

HTML 문서 내부에 같은 태그를 모두 선택할 때 사용한다.

3. 아이디 선택자

HTML 문서 내부에 해당하는 아이디 속성값을 가진 태그만 선택할 때 사용한다.

  1. 아이디 선택자 테스트 1
  2. 아이디 선택자 테스트 2
  3. 아이디 선택자 테스트 3
  4. 아이디 선택자 테스트 4

4. 클래스 선택자

HTML 문서 내부에 해당하는 클래스 속성값을 가진 태그들을 선택할 때 사용한다.

  • 클래스 선택자 테스트 1
  • 클래스 선택자 테스트 2
  • 클래스 선택자 테스트 3
  • 클래스 선택자 테스트 4
  • 클래스 선택자 테스트 5
