font-family

텍스트의 글꼴을 지정할 때 사용한다.

font-size

텍스트의 크기를 변경할 때 사용한다.

  • px/pt
  • em/rem 자식 요소
  • % 테스트

font-weight

텍스트의 두께를 변경할 때 사용한다.

  • 원래 굵기
  • 굵은 글꼴
  • 원래 굵기보다 더 가늘게
  • 원래 굵기보다 더 굵게

font-style

텍스트의 기울임을 지정할 때 사용한다.

  • italic 기울임
  • oblique 기울임

color

텍스트의 색상을 지정할 때 사용한다.

  • 색상명으로 지정
  • 16진수 값으로 지정
  • rgb/rgba로 지정
  • hsl/hsla로 지정
